# adaptive-autosar-speed-monitor

A hands-on portfolio project demonstrating AUTOSAR Adaptive Platform concepts
using vsomeip (SOME/IP) on Ubuntu Linux.

## Stack
- C++17
- CMake
- vsomeip 3.x
- Ubuntu 22.04

## Concepts Demonstrated
- Service-Oriented Architecture (SOA)
- SOME/IP protocol and Service Discovery
- ara::com communication patterns

## Milestone 0 — Environment Setup
- Installed Ubuntu 22.04 on UTM VM
- Built and installed vsomeip 3.4.10 from source
